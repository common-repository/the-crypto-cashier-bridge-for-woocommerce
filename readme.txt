The Crypto Cashier bridge for WooCommerce adds a payment gateway to The Crypto Cashier payments system. 

== Description ==

If you want to use the web shopping features of WooCommerce and also allow your customers to check-out and pay using alternative payment methods, such as crypto-currencies, you can use this bridge plugin to allow WooCommerce to interact with The Crypto Cashier plugin.

https://vimeo.com/333251643

== Installation ==

= Minimum Requirements =

* WordPress version 4.7 or greater
* PHP version 5.4 or greater (PHP 5.6 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)

= Optional Requirements =

The following components are optional. If they are not installed or not activated, then this bridge is ignored by WordPress.

* WooCommerce WordPress plugin 3.2 or greater (WooCommerce 3.3.4 is recommended)
* The Crypto Cashier WordPress plugin 1.3 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of The Crypto Cashier bridge for WooCommerce plugin, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “The Crypto Cashier bridge” and click Search Plugins. Once you’ve found the plugin you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading The Crypto Cashier bridge for WooCommerce plugin, which is a software package contained in a zip file.  Once downloaded, the zip file needs to uploaded to your webserver via FTP or SFTP (your hosting provider can provide more details on how to upload files). The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
